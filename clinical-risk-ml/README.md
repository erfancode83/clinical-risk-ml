# Clinical Risk Prediction (Machine Learning)

A production-ready machine learning pipeline for predicting clinical risk scores using patient health metrics.

## Features
- End-to-end ML pipeline using scikit-learn
- Polynomial feature engineering
- Ridge regression with cross-validation
- Model explainability (coefficients & partial dependence)
- Error analysis by clinical risk groups
